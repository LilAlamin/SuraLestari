# 🌿 SuraLestari — Eco Journey Surakarta

**SDGs Creative Web Competition - BytesFest 2026**
Sub-tema: *Eco Journey: Sustainable Tourism Experience*

## Tentang

SuraLestari adalah landing page interaktif untuk destinasi ekowisata di Surakarta. Website ini mengedukasi pengunjung tentang pelestarian lingkungan, mempromosikan UMKM lokal, dan menyediakan pengalaman wisata digital yang menarik.

## Tech Stack

- **Framework:** Next.js (App Router, React 19, TypeScript)
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui (Radix primitives)
- **Icons:** Lucide React
- **Fonts:** Roboto Serif, Roboto

## Fitur

- ✨ Hero section dengan video parallax dan efek cloud overlay
- 🗺️ Informasi destinasi ekowisata Surakarta
- 🎨 Pendekatan wisata: Alam, Budaya & Heritage, UMKM & Komunitas
- 📱 Fully responsive (mobile-first)
- 🎭 Scroll-driven animations & sticky overlapping sections

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Struktur Proyek

```
src/
  app/              # Next.js routes & layout
  components/       # React components
    ui/             # shadcn/ui primitives
  lib/
    utils.ts        # Utility functions
public/
  images/           # Aset gambar
  videos/           # Aset video
```
